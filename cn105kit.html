ขั้นตอน

สำหรับ CN105Kit

  1. ปรับสวิตช์บนโมดูล CN105Kit เป็น Flash Mode
  2. เชื่อมต่อ USB to Serial เข้ากับโมดูล CN105Kit ตามแผนภาพนี้
  3. เลือกเวอร์ชั่น Hardware ด้านล่างแล้วกดปุ่ม Connect
  4. เลือก Serial Device และทำตามขั้นตอน
  5. รอจน flash firmware สำเร็จ และทำการปรับสวิทช์โมดูลไปที่ Normal Mode
การเชื่อมต่อ USB to Serial สำหรับ CN105Kit V2 และ 2.1

อัปเดตผ่าน OTA

ขั้นตอนสำหรับ CN105Kit V1, V2.1 (ESP12-E) และ WiFiKit Serial (ESP32s3)

สามารถอัปเดตไปรุ่นถัดไปได้ทันที

  1. Download ไฟล์ไฟล์เฟิร์มแวร์ของ Mitsubishi2MQTT เวอร์ชั่นล่าสุด จากที่นี่ (เลือกไฟล์ .bin ตาม hardware ที่ใช้)
  2. เข้าไปที่หน้าเว็บ config ผ่าน IP Address ของโมดูล
  3. กด Firmware Update แล้วเลือกไฟล์ .bin ที่ดาวน์โหลดมา
  4. กด Start upgrade

ขั้นตอนสำหรับ CN105Kit V2 (ESP07) หรือรุ่นที่เจอ Error: File upload buffer miscompare

เนื่องจากชิป ESP07 มีพื้นที่ความจุ 1 MB จึงไม่สามารถอัปโหลด Firmware เวอร์ชั่นใหม่ไปเก็บในบอร์ดได้ ต้องอัปเดตเป็นเฟิร์มแวร์เวอร์ชั่น Minimal ซึ่งใช้พื้นที่น้อยก่อน แล้วจึงค่อย OTA เพื่อ Update เป็นเวอร์ชั่นล่าสุด

  1. Download ไฟล์ไฟล์เฟิร์มแวร์ของ Mitsubishi2MQTT เวอร์ชั่นล่าสุด จากที่นี่ (เลือกไฟล์ .bin ตาม hardware ที่ใช้)
  2. Download ไฟล์ไฟล์เฟิร์มแวร์ Minimal สำหรับใช้ OTA จากที่นี่
  3. เข้าไปที่หน้าเว็บ config ผ่าน IP Address ของโมดูล
  4. กด Firmware Update แล้วเลือกไฟล์เฟิร์มแวร์ Minimal ที่ดาวน์โหลดมา รอจนการติดตั้งเฟิร์มแวร์ minimal เสร็จสบมูรณ์ (LED กระพริบทุก 1 วินาที)
  5. เชื่อมต่อ Wi-Fi ของคอมพิวเตอร์ไปที่สัญญาณ Wi-Fi ที่ตัวโมดูลปล่อยออกมา (AP Mode), Wi-Fi ชื่อ esp8266-webupdate
  6. เข้าไปที่หน้าเว็บสำหรับการอัปเดต: http://192.168.4.1/update
  7. เลือกเฟิร์มแวร์ Mitsubishi2MQTT ที่ดาวน์โหลดมา แล้วทำการกด Update Firmware
  8. รอจนเสร็จสิ้น แล้วทำการตั้งค่าโมดูลใหม่

*การตั้งค่าของโมดูลจะสูญหาย โปรดจดชื่อ MQTT Hostname และค่าต่าง ๆ ไว้ก่อน (ยกเว้นรุ่น ESP12-E การตั้งค่ายังอยู่)

**หากฟีเจอร์ใหม่ไม่ปรากฎใน MQTT Integration ให้ทำการลบอุปกรณ์ออกจาก MQTT Integration แล้วทำการ Reboot Module หนึ่งรอบ
